= Launcher relational database capability

:parent-context: {context}

The {name-example-crud} capability extends the {name-example-http-api} application to provide a basic example of performing _create_, _read_, _update_ and _delete_ (_CRUD_) operations on a PostgreSQL or a MySQL database using a simple HTTP API. _CRUD_ operations are the four basic functions of persistent storage used when developing an HTTP API that interacts with a database.
