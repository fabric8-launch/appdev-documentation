= Launcher capabilities

:parent-context: {context}

When you create a new application using link:{link-launcher-oso}[{name-launcher-oso}], you can add the basic building blocks of cloud-based applications. The link:{link-launcher-oso}[{name-launcher-oso}] refers to these building blocks as capabilities. Available capabilities include a health check, a REST API, and a relational database backend.
