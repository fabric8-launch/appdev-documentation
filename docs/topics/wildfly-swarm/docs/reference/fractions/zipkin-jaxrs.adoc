= Zipkin - JAXRS Support


.Maven Coordinates
[source,xml]
----
<dependency>
  <groupId>org.wildfly.swarm</groupId>
  <artifactId>zipkin-jaxrs</artifactId>
</dependency>
----

.Configuration

swarm.zipkin.name:: 
The service name used in reports

swarm.zipkin.rate:: 
The reporting rate

swarm.zipkin.url:: 
URL of the Zipkin server
