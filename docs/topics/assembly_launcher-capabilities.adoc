= Launcher capabilities

:parent-context: {context}

When you create a new application using link:{link-launcher-oso}[{name-launcher-oso}], you can add the basic building blocks of cloud-based applications. The link:{link-launcher-oso}[{name-launcher-oso}] refers to these building blocks as capabilities. Applications created with the link:{link-launcher-oso}[{name-launcher-oso}] automatically include a health check capability. You can also add REST API or relational database backend capabilities.
