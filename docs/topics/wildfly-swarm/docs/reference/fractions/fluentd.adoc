= Fluentd


.Maven Coordinates
[source,xml]
----
<dependency>
  <groupId>org.wildfly.swarm</groupId>
  <artifactId>fluentd</artifactId>
</dependency>
----

.Configuration

swarm.fluentd.hostname:: 
Host name of the fluentd server

swarm.fluentd.level:: 
Logging level

swarm.fluentd.port:: 
Port of the fluentd server

swarm.fluentd.tag:: 
Logging tag
