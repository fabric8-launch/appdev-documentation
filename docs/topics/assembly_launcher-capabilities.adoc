= Launcher capabilities

:parent-context: {context}

When you create a new application using link:{link-launcher-oso}[{name-launcher-oso}], you can add the basic building blocks for cloud-based applications. 
The link:{link-launcher-oso}[{name-launcher-oso}] refers to these building blocks as capabilities. 
Available capabilities include:

* Health check
* REST API
* Relational database backend
